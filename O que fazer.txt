implemente uma API em Node.Js e persistência no SqlServer Express, que possibilite salvar os seguintes dados de cadastro de produtos:

Nome do produto;
Código do produto;
Preço;
Descrição;
Quantidade em estoque;
Avaliação; e
Categoria.

Utilize as ações do CRUD para criar, ler, editar e deletar dados do banco SqlServer Express. Desta forma, a sua API deve ter rotas para:

Criação de dados com Post;
Leitura de todos os dados do banco com Get;
Leitura de dados por código do produto com Get;
Leitura de dados por categoria com Get;
Atualização de informações com Put ou Patch; e
Exclusão de elementos com Delete.

 Você desenvolverá um serviço Web similar ao desenvolvido nos vídeos, com duas páginas. A primeira página será a home e a segunda a tela de detalhes para apresentar:

Na página principal (Home)

A coleção dos últimos 12 itens salvos no banco de dados com imagem e preço;
Um botão em cada item que possibilite ir para uma nova página com os detalhes do produto; e
Um botão com um modal (ou nova página) para acrescentar novos itens de forma dinâmica.
Na página de detalhes

Apresentar as todas as informações salvas do produto selecionado;
Um botão com um modal (ou nova página)  para editar o item; e 
Um botão para excluir o item com solicitação de confirmação de exclusão.
É recomendável utilizar Bootstrap com javascript e css para criar seus próprios estilos, mas, caso tenha mais facilidade, você poderá utilizar somente html (puro), css e javascript.
